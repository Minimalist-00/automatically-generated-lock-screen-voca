# 自動生成ロック画面単語帳 (Lock Screen Voca) 要件定義書

毎日英単語を学習するために、スマートフォンのロック画面へその日に設定した3つの単語を表示する画像を生成するアプリケーション。

## 1. システム構成
- **フロントエンド**: Next.js, TypeScript
- **データベース / バックエンド**: Supabase
- **AI 連携**: Gemini API（単語カードのコンテンツ自動生成）

## 2. 機能要件

### 2.1. 単語カードの作成・管理
- **手動登録**: 単語と日本語の意味を入力して簡易登録できる（クイックメモ機能）。
- **AI自動生成**: Gemini APIを利用し、登録された単語から以下の項目を自動生成する。
  - 使えるシーン
  - 例文
- **コンテンツ構成**:
  - 単語
  - 意味
  - 使えるシーン
  - 例文

### 2.2. 単語リスト表示
- 登録されている単語の一覧を表示・管理する機能。

### 2.3. 壁紙画像の管理
- ユーザーが独自の「元の壁紙」をアップロードし、保存・選択できる機能。

### 2.4. ロック画面画像生成システム
- **単語の選定**: リストから「今日の単語」として3つ選択し、クエストとして設定する機能。
- **画像の生成**: 選択した「元の壁紙」に、選んだ3つの単語カードをレイアウトしたロック画面用画像を生成する機能。
- **UI/デザイン考慮**:
  - iPhone 16 / iPhone 17 の画面比率（iPhone 17の仕様を参照）。
  - 時計やウィジェットの表示位置を避けたレイアウト。
  - 元の壁紙に重ねても文字が見やすくなる工夫（例：半透明のグラスモフィズム風のカードで囲むなど、モダンでプレミアムなデザイン）。

## 3. 非機能要件
- **ユーザー認証**: なし（自分専用のシングルユーザー環境を想定）。
- **レスポンシブ対応**: 画像生成および管理画面はPC/スマートフォンの両方で操作しやすいUI。

## 4. ユーザー設定・環境構築要件
アプリケーションの動作に必要な以下の設定情報は、実装完了後に開発者から提示され、ユーザー側で設定を行います。
- **Gemini API**: APIキーの取得と環境変数の設定。
- **Supabase**: プロジェクトの作成、接続情報（URL, Anon Key）の環境変数設定、およびデータベース（単語・壁紙用テーブル）のスキーマ適用。
